# Getting Started with Authentication and Authorization in .NET

This repository contains starter files and complete sample solutions for the course  
**Getting Started with Authentication and Authorization in .NET**, available on [Dometrain](https://dometrain.com/).

## Included Projects  
The following Visual Studio projects are provided as part of the course: 

* **Authentication Project**  
  Source code used throughout the authentication sections of the course.
* **Authorization Project**  
  Demonstrates the concepts and implementations covered in the authorization chapters.
* **MVC Filters Project**  
  Example code for the MVC Filter Pipeline module.
